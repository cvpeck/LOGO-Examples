Deutsch

Durch Verwendung des Templates des LOGO! TD können Sie sich Ihre eingene Dokumentation der Menüstruktur erstellen.
1) Öffnen Sie das Bild des LOGO! TD mit einem Bearbeitungsprogramm (z.B. Paint, Word).
2) Starten Sie die Simulation von LOGO! Soft Comfort. 
3) Machen Sie Screenshots der entsprechenden Meldetexte (z.B. mit der Taste Druck). 
4) Fügen Sie das Bild, das sich nun in der Zwischenablage befindet, in das Bearbeitungsprogramm ein (Strg+V).



English

By using the template of LOGO! TD you are able to create your own documentation of the menu structure.
1) Open the image of the LOGO! TD by using a image editing program (e.g. Paint, Word).
2) Start the simulation of LOGO! Soft Comfort.
3) Make screenshots of the corresponding message texts (e.g. with print key).
4) Insert the picture, which is now in the clipboard, into the image editing program (Strg+V).
